# FileBrowser - 文件管理器

> 轻量级 Web 界面文件管理器，支持文件上传、下载、分享和在线预览。
>
> - **官方文档**：[FileBrowser Docs](https://filebrowser.org/installation.html)

---

## 一、 部署架构 (Deployment)

FileBrowser 采用单容器桥接部署（Bridge），不干预底层网络。

### 1.1 容器架构说明

**单容器服务**：
- **filebrowser**：Web 文件管理服务，提供浏览器界面的文件上传、下载、分享和在线预览功能。

### 1.2 路径映射逻辑

- **Service Local (服务自产层)**：
  - `./database` 目录：包含 `filebrowser.db` 核心数据库，保存用户配置及分享链接鉴权信息。
  - `./config` 目录：包含部分可覆盖服务设定的静态配置文件（如需）。
- **Resource (媒体资源层)**：
  - `${CLOUD_PATH}`：挂载为 `/srv`。赋予 FileBrowser 查阅该海量资源层并对外提供 Web 可视化的能力。

## 二、 快速启动 (Quick Start)

> [!IMPORTANT]
> **资源路径检查**：部署前，请确保 `global.env` 中 `${CLOUD_PATH}` 变量已定义并正确指向目标物理路径（如用户私有云资源池）。

1. 查阅与验证配置逻辑：
   - 配置文件：[FileBrowser.yml](../../../Compose/FileBrowser/FileBrowser.yml)
2. 在 OMV Web UI 的 Compose 标签页，加载此 `FileBrowser.yml` 文件。
3. 点击 `Up`（启动）即可。管理员账号为 `admin`，初始管理员密码通过查看 `docker logs <container_name_or_id>` 获取。

## 三、 详细配置参数表 (Configuration Table)

所有环境变量声明于 `FileBrowser.yml`，且严格依赖宿主机的 `global.env` 透传。

| 选项 | 值/变量 | 修改建议 | 说明 |
| :--- | :--- | :--- | :--- |
| **基础配置** | | | |
| 容器映像 | `filebrowser/filebrowser:v2-s6` | 🟢 保持默认 | s6 镜像天然支持并严格执行用户权限降级 (`PUID`) |
| 端口映射 | `"8081:80"` | 🟡 按需调整 | 将容器内置侦听的 `80` 暴露于 OMV 的 `8081` 端口 |
| 外部网络模式 | 无 (Bridge) | 🟢 保持默认 | 默认通过 Bridge 桥接通讯，避免 host 端口碰撞 |
| **环境变量 (Environment)** | | | |
| 权限 UID | `PUID` | 🟢 保持默认 | 指向 `${PUID}`（OMV `appuser` 的数值 1001） |
| 权限 GID | `PGID` | 🟢 保持默认 | 指向 `${PGID}`（`users` 组的数值 100） |
| 时区同步 | `TZ` | 🟢 保持默认 | 指向 `${TZ}`（如 `Asia/Shanghai`），防止时间戳错位 |
| 容器内部端口 | `FB_PORT: 80` | 🟢 保持默认 | 改变容器内启动端口，与 Bridge 的端口映射对齐 |
| 数据库寻址 | `FB_DATABASE` | 🟢 保持默认 | 设定内置建库路径 `/database/filebrowser.db` |
| 资源根目录 | `FB_ROOT: /srv` | 🟢 保持默认 | 限定主界面的读取起始位置 |
| **目录挂载 (Volumes)** | | | |
| 主机时间挂载 | `${LOCALTIME_PATH}` | 🟢 保持默认 | `/etc/localtime` 确保日志的时间属性严格一致 |
| 数据库存储 | `./database` | 🟢 保持默认 | 映射至容器 `./database` |
| 配置目录存储 | `./config` | 🟢 保持默认 | 映射至容器 `./config` |
| 核心资源盘 | `${CLOUD_PATH}` | 🔴 必须对应 | 提供云端数据文件的本体读取权限 |

## 四、 首次登录配置 (Initial Setup)

首次通过 `IP:8081` 登录后，需完成以下 Settings 配置：

- **个人设置**
  - 个人设置
    - 不显示隐藏文件 → 勾选
    - 语言 → 中文
  - Change Password → 修改密码
- **全局设置**
  - 全局设置
    - 在添加新用户的同时自动创建用户的主目录 → 勾选
  - 用户默认设置
    - 语言 → 中文
- **用户管理**
  - 新建用户
